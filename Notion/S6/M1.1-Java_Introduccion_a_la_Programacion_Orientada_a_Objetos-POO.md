# Java - Introducción a la Programación Orientada a Objetos (POO)

## 📚 Introducción a la Programación Orientada a Objetos (POO)

## 🧠 ¿Qué es la Programación Orientada a Objetos (POO)?

La **Programación Orientada a Objetos (POO)** es un paradigma de programación que organiza el código en **objetos**. Cada objeto representa una **entidad real** (como un auto, un perro, un usuario) que tiene **estado** y **comportamiento**.

En POO, las **clases** actúan como **plantillas** o **modelos** que definen cómo deben ser esos objetos y qué acciones pueden realizar. Al crear un objeto, estamos **instanciando** una clase, creando una "copia real" de esa plantilla.

## 🛠️ ¿Qué es un Objeto en Java?

Un **objeto** es una abstracción que representa un **entidad del mundo real** que tiene **estado** y **comportamiento**. Los objetos permiten modelar situaciones reales en el mundo del software.

**Ejemplo:**

Imagina que quieres modelar vehículos en un programa. Podrías tener una **clase Vehículo**, y **objetos** como **Auto**, **Camión**, **Bicicleta** que serán creados a partir de esa clase.

## 🔑 ¿Qué es una Clase?

Una **clase** es un molde o una plantilla que define las características (atributos) y comportamientos (métodos) de un objeto. En términos simples, una clase describe **cómo debe ser un objeto** (su estado) y **qué puede hacer** (su comportamiento).

## 🏗️ ¿Cómo se compone una Clase?

Veamos los componentes de una clase con un ejemplo de **Perro**.

### 📋 Ejemplo de Clase Perro

```java
// Declaración de la clase Perro
public class Perro {

    // Atributos (Estado) - Las características del perro
    // Cada atributo representa una caracteristica del objeto
    String nombre;
    String raza;
    int edad;

    // Métodos (Comportamiento) - Acciones que el perro puede hacer
    // Los metodos son las acciones que puede realizar un objeto.
    public void ladrar() {
        System.out.println("¡Guau! ¡Guau!");
    }

    // Método para imprimir información sobre el perro
    public void imprimirInformacion() {
        System.out.println("Nombre: " + nombre);
        System.out.println("Raza: " + raza);
        System.out.println("Edad: " + edad + " años");
    }
    
    // Constructor - Inicializa los atributos
    public Perro(String nombre, String raza, int edad) {
        this.nombre = nombre;
        this.raza = raza;
        this.edad = edad;
    }
}
```

## 🚀 ¿Cómo Crear un Objeto?

Una vez que tienes la clase definida, puedes crear **instancias** de esa clase (es decir, crear objetos) usando la palabra clave `new`. Al instanciar un objeto, **se llama al constructor** para inicializarlo.

**Ejemplo de Creación de un Objeto**

```java
// Crear un objeto Perro llamado "Rex"
Perro perro1 = new Perro("Rex", "Pastor Alemán", 4);

// Llamamos a los métodos del objeto
perro1.ladrar();
perro1.imprimirInformacion();
```

## 📦 ¿Qué es un Package (Paquete)?

En Java, un **package** (paquete) es una forma de organizar el código y evitar que se repitan los nombres de las clases en el proyecto. Los paquetes actúan como **carpetas** que agrupan clases relacionadas.

**¿Por qué usar paquetes?**

1. Organizan el código y facilitan la estructura del proyecto.
2. Permiten evitar conflictos de nombres entre clases.
3. Reflejan la estructura del sistema de archivos.

**Ejemplo de Declaración de Paquete:**

```java
package com.miproyecto.animales;  // Paquete donde se encuentra la clase Perro
```

## 🧩 Ejercicio Guiado: Creando una Clase y Objetos

### 🎯 Objetivo

Vamos a crear nuestra propia clase **Estudiante** y utilizarla en el `main` para ver cómo se instancian los objetos. Los estudiantes podrán crear sus atributos (como nombre, edad, etc.) y un método que permita imprimir sus datos.

### Pasos:

1. Crea una clase llamada **Estudiante** con los atributos:
    - `nombre` (String)
    - `edad` (int)
    - `carrera` (String)
2. Añade un **constructor** para inicializar estos atributos.
3. Crea un **método** llamado `imprimirDatos()` que imprima la información del estudiante en consola.
4. En el método `main`, **instancia al menos dos objetos Estudiante**, asigna valores a sus atributos y usa el método `imprimirDatos()`.

### Código Ejemplo

```java
public class Estudiante {

    // Atributos
    String nombre;
    int edad;
    String carrera;

    // Constructor
    public Estudiante(String nombre, int edad, String carrera) {
        this.nombre = nombre;
        this.edad = edad;
        this.carrera = carrera;
    }

    // Método para imprimir los datos
    public void imprimirDatos() {
        System.out.println("Nombre: " + nombre);
        System.out.println("Edad: " + edad + " años");
        System.out.println("Carrera: " + carrera);
    }

    public static void main(String[] args) {
        // Crear objetos Estudiante
        Estudiante estudiante1 = new Estudiante("Juan Pérez", 20, "Ingeniería Informática");
        Estudiante estudiante2 = new Estudiante("María López", 22, "Derecho");

        // Llamar a imprimirDatos
        estudiante1.imprimirDatos();
        estudiante2.imprimirDatos();
    }
}
```

## 🧠 Conceptos a Reforzar:

- **Clase:** Plantilla que define los atributos y métodos.
- **Objeto:** Instancia de una clase creada con `new`.
- **Atributos:** Variables dentro de la clase que describen el estado del objeto.
- **Métodos:** Funciones dentro de la clase que definen lo que el objeto puede hacer.
- **Constructor:** Método especial que se usa para inicializar los objetos cuando se crean.
