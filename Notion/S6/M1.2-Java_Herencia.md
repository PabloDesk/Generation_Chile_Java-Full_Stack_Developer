# Java - Herencia

## 🎯 Objetivo

Aprender a **reutilizar código** creando clases a partir de otras, evitando duplicación y organizando mejor nuestro modelo.

Hasta ahora:

- Sabemos crear clases
- Sabemos encapsular atributos
- Sabemos instanciar objetos

Ahora damos el siguiente paso: **heredar**.

---

## 🧠 ¿Qué es la Herencia?

|La **herencia** permite crear una nueva clase a partir de una clase existente.

La nueva clase:

- **Hereda atributos**
- **Hereda métodos**
- Puede **agregar cosas nuevas**
- Puede **modificar comportamientos** (más adelante)

📌 En Java se usa la palabra clave: `extends`

---

## 👪 Relación padre → hijo

- **Clase padre (superclase):** define lo común
- **Clase hija (subclase):** especializa

Ejemplo conceptual:

```
Animal
 ├─ nombre
 ├─ edad
 └─hacerSonido()

Gato
 └─ vidas

Perro
 └─ raza
```

---

## 🐾 Ejemplo base: Animal → Gato

### Clase padre: `Animal`

```java
class Animal {

		protected String nombre;
		protected int edad;

    Animal(String nombre,int edad) {
			this.nombre = nombre;
			this.edad = edad;
    }

		void comer() {
        System.out.println(nombre +" está comiendo 🍖");
    }
}

```

📌 `protected` permite:

- Acceso desde la clase hija
- Sin exponer los atributos públicamente

---

### Clase hija: `Gato`

```java
class Gato extends Animal {

		int vidas;

    Gato(String nombre,int edad,int vidas) {
			super(nombre, edad);// llama al constructor del padre
			this.vidas = vidas;
    }

		void maullar() {
        System.out.println(nombre +" dice: miau 🐱");
    }
}

```

---

### Clase `App`

```java
public class App {

public static void main(String[] args) {

	      Gato gato = new Gato("Michi",2,7);

				// Atributos heredados
        System.out.println(gato.nombre);
        System.out.println(gato.edad);

				// Atributo propio
        System.out.println("Vidas: " + gato.vidas);

				// Métodos
        gato.comer();// heredado
        gato.maullar();// propio
    }
}

```

---

## 🔍 Qué está pasando aquí (clave para explicar)

| Elemento | ¿De dónde viene? |
| --- | --- |
| `nombre` | Clase Animal |
| `edad` | Clase Animal |
| `comer()` | Clase Animal |
| `vidas` | Clase Gato |
| `maullar()` | Clase Gato |

👉 **El objeto `gato` tiene TODO eso junto**.

---

## 🧠 Idea clave de herencia

> La clase hija ES un tipo de la clase padre
> 

Un `Gato`:

- **ES un** `Animal`
- Pero un `Animal` **NO necesariamente** es un `Gato`

---

## ❌ Errores típicos a prevenir (muy importante)

❌ Copiar y pegar código en vez de heredar

❌ Crear herencias solo “porque sí”

❌ Heredar cosas que no tienen sentido

Ejemplo incorrecto:

```
Auto extends Perro ❌
```

---

## 🧩 Herencia no es reemplazo total

- El padre define lo común
- El hijo agrega lo específico

---

Gato.java

```java
package herencia;

//Gato hereda todos de mascota pero tiene sus propios atributos
public class Gato extends Mascota {

    boolean esIndoor; // true si vive solo dentro de la casa

    public Gato(String nombre, int edad, boolean esIndoor) {
        super(nombre, edad); //     inicializa nombre y edad de padre
        this.esIndoor = esIndoor;
    }

    public void maullar(){
        System.out.println(nombre + " dice: !Miau¡");
    }
}
```

---

Main.java

```java
package herencia;

public class Main {


    public static void main(String[] args) {
        //Perro
        Perro perro = new Perro("Pedro", 5, "Quiltro");
        perro.presentarse(); // Al padre
        perro.ladrar(); // metodo solo del hijo


        //Gato
        Gato gato = new Gato("Bola de nieve", 2, true);
        gato.presentarse();
        gato.maullar();
    }

}
```

---

Mascota.java

```java
package herencia;

public class Mascota {

    //Atributos los compartiremos con cualquier mascota
    String nombre;
    int edad;

    //Constructor Padre
    public Mascota(String nombre, int edad) {
        this.nombre = nombre;
        this.edad = edad;
    }

    public void presentarse(){
        System.out.println("Hola, soy " + nombre
                + " de " + edad + " años. ");
    }
}
```

---

Perro.java

```java
package herencia;

public class Perro extends Mascota{

    String raza;

    // Constructor de Perro: recibe todos los datos de mascota
    public Perro(String nombre, int edad, String raza) {
        super(nombre, edad);
        this.raza = raza;
    }

    // su propio metodo
    public void ladrar(){
        System.out.println(nombre + " dice : !Guau¡");
    }
}
```
