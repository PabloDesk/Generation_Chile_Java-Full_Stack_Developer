# Ejercicio guiado de POO en Java (visibilidad, herencia, encapsulación)

## 🎯 **Objetivo**

Crear un pequeño programa en Java que modele publicaciones (Libros y Revistas), usando **POO básica**, **modificadores de acceso** y **herencia**.

Al final, extenderás el ejercicio con **Scanner** y **colecciones**.

---

---

## ✅ **PASO 1 — Crear el proyecto**

1. Abre tu IDE favorito (IntelliJ, Eclipse o VS Code).
2. Crea un nuevo proyecto Java.
3. Crea la estructura de paquetes:

```
src/
├── modelo/
└── servicios/
```

---

## 🧱 **PASO 2 — Crear la clase Publicacion**

📌 **Ubicación:** `src/modelo/Publicacion.java`

### 1. Crear clase y atributos:

```java
package modelo;

public class Publicacion {
    String titulo;// default
		protected String autor;// protected
		private int anio;// private
}
```

### 📝 **Explicación**

| Modificador | Visibilidad |
| --- | --- |
| `public` | En cualquier parte del proyecto |
| *default* | Solo dentro del mismo paquete |
| `protected` | Dentro del paquete + subclases |
| `private` | Solo dentro de la clase |

---

## 💡 **PASO 3 — Encapsular atributos**

Agrega un constructor y **getters/setters**:

```java
public Publicacion(String titulo, String autor, int anio) {
	this.titulo = titulo;
	this.autor = autor;
	this.anio = anio;
}

public int getAnio() {
	return anio;
}

public void mostrarInformacion() {
    System.out.println("Título: " + titulo);
    System.out.println("Autor: " + autor);
    System.out.println("Año: " + anio);
}
```

🔍 **Tarea para el alumno:**

- ¿Puedes acceder directamente a `anio` desde otra clase?
    
    -> Prueba en `main` e intenta usar `publicacion.anio`. ¿Da error? ¿Por qué?

## 🧬 **PASO 4 — Crear subclases**

### 📄 `Libro.java`

```java
package modelo;

public class Libro extends Publicacion {
	  private int paginas;

		public Libro(String titulo, String autor, int anio,int paginas) {
			super(titulo, autor, anio);
			this.paginas = paginas;
    }

		@Override
		public void mostrarInformacion() {
				super.mostrarInformacion();
        System.out.println("Páginas: " + paginas);
    }
}
```

### 📄 `Revista.java`

```java
package modelo;

public class Revista extends Publicacion {
		private int numero;

		public Revista(String titulo, String autor, int anio,int numero) {
				super(titulo, autor, anio);
				this.numero = numero;
    }

		@Override
		public void mostrarInformacion() {
				super.mostrarInformacion();
        System.out.println("Número de revista: " + numero);
    }
}

```

📝 **Prueba de acceso protegido**

- Desde una subclase `Libro`, intenta acceder a `autor`.
- ¿Puedes hacerlo? Sí → porque es `protected`.

## 🚀 **PASO 5 — Crear la clase GestorBiblioteca**

📌 **Ubicación:** `src/servicios/GestorBiblioteca.java`

```java
package servicios;

import modelo.Publicacion;
import java.util.ArrayList;

public class GestorBiblioteca {
		private ArrayList<Publicacion> publicaciones =newArrayList<>();

		public void agregarPublicacion(Publicacion p) {
        publicaciones.add(p);
    }

		public void mostrarTodo() {
				for (Publicacion p : publicaciones) {
	          System.out.println("--------");
            p.mostrarInformacion();
        }
    }
}
```

🔍 **Nota:**

Aquí usamos un `ArrayList` para guardar publicaciones.

👉 Verás por qué usamos la **superclase** `Publicacion` como tipo general.
