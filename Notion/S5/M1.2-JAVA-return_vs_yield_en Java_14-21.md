# JAVA - `return` vs `yield` en Java (Java 14 → 21)

## 🎯 Objetivo

Entender qué hace `return` y  `yield` cuándo usarlo y cómo mejora la lógica del código.

---

# 🟢 `yield` en Java (Switch moderno Java 14 → 21)

`yield` se usa **solo dentro del `switch` moderno** para **devolver un valor desde un bloque `{}`**.

Se utiliza cuando el `case` tiene **más de una línea**.

---

## 🎯 Ejemplo básico con `yield`

```java
int opcion = 2;

String resultado = switch (opcion) {
    case 1 -> {
        yield "Opción uno";
    }
    case 2 -> {
        yield "Opción dos";
    }
    default -> {
        yield "Otra opción";
    }
};

System.out.println(resultado);
```

Aquí:

- cada `case` devuelve un valor
- el switch guarda ese valor en `resultado`
- el programa continúa normalmente

---

## 🧠 Cuándo usar `yield`

Usar `yield` cuando el `case` tiene lógica interna:

```java
int opcion = 2;

String resultado = switch (opcion) {
    case 1 -> {
        yield "Opción uno";
    }
    case 2 -> {
        yield "Opción dos";
    }
    default -> {
        yield "Otra opción";
    }
};

System.out.println(resultado);
```

Aquí:

- primero ejecuta `println`
- luego `yield` devuelve el valor

---

## ❌ Sin `yield` (error)

```java
case 1 -> {
    "Hola";
}
```

Esto no compila.

Debe ser:

```java
case 1 -> {
    yield "Hola";
}
```

---

## ✔ Cuando NO necesitas `yield`

Si el `case` es corto:

```java
String resultado = switch (opcion) {
    case 1 -> "Uno";
    case 2 -> "Dos";
    default -> "Otro";
};
```

Aquí no se usa `yield`.

---

# 🔁 `return` vs `yield`

## Diferencia principal

| Palabra | Qué hace |
| --- | --- |
| `return` | Termina el método completo |
| `yield` | Devuelve valor dentro del switch |

---

## 🔙 `return`

`return` sale del método completamente.

```java
public static String test() {

    return "Hola";

    // nunca llega aquí
}
```

Después de `return`, nada se ejecuta.

---

## 🔁 `yield`

`yield` solo devuelve valor al switch, **no termina el método**.

```java
String texto = switch (x) {
    case 1 -> {
        yield "Hola";
    }
    default -> {
        yield "Adiós";
    }
};

System.out.println("El programa continúa");
```

Aquí el programa sigue.

---

# 🔥 Comparación directa

## Con `return`

```java
String texto = switch (x) {
    case 1 -> {
        yield "Hola";
    }
    default -> {
        yield "Adiós";
    }
};

System.out.println("El programa continúa");
```

Aquí el método termina.

---

## Con `yield`

```java
String resultado = switch (x) {
    case 1 -> {
        yield "Uno";
    }
    default -> {
        yield "Otro";
    }
};

System.out.println("Continúa ejecución");
```

Aquí el método continúa.

---

# 🧠 Regla simple

> `return` → salir del método
> 
> 
> `yield` → devolver valor del switch
> 

---

## 🎯 Resumen

|  | return | yield |
| --- | --- | --- |
| Sale del método | ✔ | ❌ |
| Devuelve valor | ✔ | ✔ |
| Solo dentro de switch | ❌ | ✔ |
| Java moderno | ✔ | ✔ |
| Continúa ejecución | ❌ | ✔ |

import java.util.Scanner;

public class CineSW {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);

        double precioBase = 5000;
        int edad;
        int categoria;
        double porcentajeDescuento = 0;
        double montoDescuento = 0;
        double precioFinal = 0;
        String nombreCategoria;

        System.out.println("=====================================");
        System.out.println("  SISTEMA DE VENTA - CINE JAVALIMOS  ");
        System.out.println("=====================================");

        System.out.print("Ingrese la edad del cliente: ");
        edad = scanner.nextInt();

        //CREAMOS UNA VALIDACION
        if (precioBase <= 0){
            System.out.println("☢️ El precio base debe ser mayor a 0.");
            scanner.close();
            return;
        }

        if (edad < 0) {
            System.out.println("☢️ El edad no puede ser negativa");
            scanner.close();
            return;
        }

        if (edad <= 5){
            categoria = 1;
        } else if (edad <=17) {
            categoria = 2;
        } else if (edad <= 65) {
            categoria = 3;
        }else {
            categoria = 4;
        }

        /*

        SWITCH ANTIGUO ANTES DE JAVA 14


        switch (categoria) {
            case 1:
                porcentajeDescuento = 50;
                nombreCategoria = "Niño pequeño";
                break;
            case 2:
                porcentajeDescuento = 30;
                nombreCategoria = "Menor de edad";
                break;
            case 3:
                porcentajeDescuento = 0;
                nombreCategoria = "Adulto";
                break;
            default:
                porcentajeDescuento = 50;
                nombreCategoria = "Adulto Mayor";
        }

        */

        // SWITCH MODERNO 14+ AL 21

        String resultado = switch (categoria){
            case 1 -> {
                porcentajeDescuento = 50;
                yield "Niño pequeño";
            }
            case 2 -> {
                porcentajeDescuento = 30;
                yield "Menor de edad";
            }
            case 3 ->{
                porcentajeDescuento = 0;
                yield "Adulto";
            }
            default -> {
                porcentajeDescuento = 50;
                yield "Adulto Mayor";
            }
        };

        nombreCategoria = resultado;

        //Calculos
        montoDescuento = precioBase * (porcentajeDescuento/100);
        precioFinal = precioBase - montoDescuento;

        System.out.println();
        System.out.println("==============BOLETA===============");

        System.out.printf("Categoria: %s%n", nombreCategoria);
        System.out.printf("Edad: %d años%n", edad);

        System.out.printf("Precio Base: $%.0f%n", precioBase);
        System.out.printf("Descuento aplicado: %.0f%%%n", porcentajeDescuento);
        System.out.printf("Monto descuento : $%.0f%n", montoDescuento);
        System.out.printf("Total a pagar: $%.0f%n",precioFinal);

        System.out.println("============== GRACIAS POR VISITARNOS ===============");


        scanner.close();
    }
}
