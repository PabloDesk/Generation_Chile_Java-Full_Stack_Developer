# Java - ARRAYS (Arreglos)

## 🎯 **Objetivo de esta página**

Al terminar esta sección, el estudiante será capaz de:

- Entender qué es un arreglo en Java
- Saber **por qué existen**
- Declarar e inicializar arrays correctamente
- Acceder y modificar sus elementos
- Recorrer arrays con distintos loops
- Evitar errores comunes
- Aplicar buenas prácticas reales
- Reconocer cuándo **NO** usar arrays

---

## 🧠 **1. ¿Qué es un Array en Java?**

Un **array (arreglo)** es una estructura que permite almacenar **varios valores del mismo tipo** bajo un mismo nombre.

Ejemplo conceptual:

> [!WARNING]
> En vez de tener muchas variables:
> 

```java
int edad1;
int edad2;
int edad3;
```

Usamos un array:

```java
int[] edades = new int[3];
```

---

## ❓ ¿Por qué son importantes los arrays?

Los arrays permiten:

✔ Agrupar datos relacionados

✔ Trabajar con loops

✔ Evitar duplicación de código

✔ Representar listas simples

✔ Preparar el camino hacia colecciones más avanzadas

Son el **primer paso real** para trabajar con múltiples datos.

---

## 🧩 **2. Características Clave de los Arrays**

| Característica | Descripción |
| --- | --- |
| Tamaño fijo | No puede cambiar después de crearse |
| Índices | Comienzan en 0 |
| Tipo único | Todos los elementos son del mismo tipo |
| Acceso rápido | Por índice |
| Estructura básica | No pertenece al Collections Framework |

---

## ✍️ **3. Sintaxis de un Array**

### Forma recomendada (moderna)

```java
int[] numeros;
```

### Forma antigua (válida pero menos usada)

```java
int numeros[];
```

✔ Ambas funcionan

✔ La primera es la **buena práctica**

---

## 🧱 **4. Declaración, Inicialización y Asignación**

### 🔹 Declarar (sin valores)

```java
int[] edades;
```

---

### 🔹 Inicializar (crear el array)

```java
edades = new int[3];
```

---

### 🔹 Declarar + inicializar

```java
int[] edades = new int[3];
```

---

### 🔹 Inicializar con valores

```java
int[] edades = {20, 25, 30};
```

---

## 🔎 **5. Acceso a los Elementos**

```java
int[] edades = {20, 25, 30};

System.out.println(edades[0]); // 20
System.out.println(edades[1]); // 25
System.out.println(edades[2]); // 30
```

📌 **Recordar:**

- Primer índice → `0`
- Último índice → `length - 1`

---

## ✏️ **6. Modificar un Elemento**

```java
edades[1] = 26;
```

Ahora el array queda:

```
[20, 26, 30]
```

---

## 🔁 **7. Recorrer un Array**

### 🔹 Con `for` (el más usado)

```java
for (int i = 0; i < edades.length; i++) {
    System.out.println(edades[i]);
}
```

✔ Control total

✔ Acceso por índice

---

### 🔹 Con `foreach` (más limpio)

```java
for (int edad : edades) {
    System.out.println(edad);
}
```

✔ Más legible

❌ No permite modificar directamente

---

### 🔹 Con `while`

```java
int i = 0;
while (i < edades.length) {
    System.out.println(edades[i]);
    i++;
}
```

---

## 🧮 **8. Propiedad `length`**

`length` indica el tamaño del array.

```java
int total = edades.length;
```

⚠️ No es un método → **no lleva paréntesis**

❌ Incorrecto:

```java
edades.length();
```

---

## 🚫 **9. Errores Comunes con Arrays**

### ❌ IndexOutOfBoundsException

```java
edades[3]; // ❌ índice inválido
```

✔ Solución:

```java
edades[edades.length - 1];
```

---

### ❌ Usar índices mágicos

```java
edades[2]; // ❌ ¿por qué 2?
```

✔ Mejor:

```java
edades[edades.length - 1];
```

---

### ❌ Olvidar inicializar

```java
int[] datos;
datos[0] = 5; // ❌ NullPointerException
```

---

## 🧼 **10. Buenas Prácticas con Arrays**

✔ Usar `length` siempre

✔ Declarar arrays cerca de su uso

✔ Inicializar todos los elementos si son críticos

✔ Usar arrays para tamaños fijos

✔ Usar nombres descriptivos

Ejemplo:

```java
int[] horasTrabajadas;
```

---

## ❌ **11. Malas Prácticas**

❌ Usar arrays cuando el tamaño cambia

❌ Usar arrays gigantes innecesarios

❌ Mezclar tipos (no permitido)

❌ Acceder sin validar límites

❌ Reemplazar List por arrays “porque es más simple”

---

## 🔄 **12. Comparación: Array vs List**

| Característica | Array | List |
| --- | --- | --- |
| Tamaño | Fijo | Dinámico |
| Flexibilidad | Baja | Alta |
| Sintaxis | Simple | Más verbosa |
| Métodos útiles | ❌ | ✔ |
| Uso inicial | ✔ | ✔ |

👉 **Empieza con arrays, evoluciona a List.**
