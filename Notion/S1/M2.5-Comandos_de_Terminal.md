# 💻 Comandos de Terminal Comunes

| Categoría | Windows (CMD / PowerShell) | Linux | macOS | Descripción Técnica |
| --- | --- | --- | --- | --- |
| 📂 **Listar archivos y carpetas** | `dir` | `ls` | `ls` | Muestra el contenido del directorio actual. Se puede extender con opciones como `ls -l` (detallado) o `ls -a` (archivos ocultos). |
| 📁 **Cambiar de directorio** | `cd <ruta>` | `cd <ruta>` | `cd <ruta>` | Cambia el directorio de trabajo actual al especificado. Soporta rutas absolutas o relativas. |
| 🧭 **Ver ubicación actual** | `cd` (sin argumentos) | `pwd` | `pwd` | Muestra la ruta absoluta del directorio en el que se encuentra el usuario. |
| 📄 **Crear un archivo** | `echo > archivo.txt` o `New-Item archivo.txt` | `touch archivo.txt` | `touch archivo.txt` | Crea un archivo vacío o actualiza la fecha de modificación si ya existe. |
| 📂 **Crear una carpeta** | `mkdir carpeta` | `mkdir carpeta` | `mkdir carpeta` | Crea un nuevo directorio en la ubicación actual. |
| 🗑️ **Eliminar archivo** | `del archivo.txt` | `rm archivo.txt` | `rm archivo.txt` | Elimina un archivo permanentemente del sistema. |
| 🧺 **Eliminar carpeta** | `rmdir /s /q carpeta` | `rm -r carpeta` | `rm -r carpeta` | Elimina un directorio y su contenido. En Linux/macOS, `-r` indica recursividad. |
| 📝 **Ver contenido de un archivo** | `type archivo.txt` | `cat archivo.txt` | `cat archivo.txt` | Muestra en pantalla el contenido de un archivo de texto. |
| 🔍 **Buscar texto dentro de archivos** | `find "texto" archivo.txt` | `grep "texto" archivo.txt` | `grep "texto" archivo.txt` | Busca coincidencias dentro de un archivo o flujo de texto. Soporta expresiones regulares. |
| 🔎 **Buscar archivos o carpetas** | `dir /s nombre` | `find . -name "archivo"` | `find . -name "archivo"` | Busca archivos o carpetas por nombre dentro del sistema de archivos. |
| 🧮 **Contar líneas o palabras** | `find /c /v "" archivo.txt` | `wc archivo.txt` | `wc archivo.txt` | Muestra el número de líneas, palabras y caracteres del archivo especificado. |
| 🧑‍💻 **Ver usuario actual** | `whoami` | `whoami` | `whoami` | Muestra el nombre del usuario activo en la sesión de terminal. |
| 🧾 **Ver procesos activos** | `tasklist` | `ps aux` | `ps aux` | Lista los procesos actualmente en ejecución. En Linux/macOS se puede combinar con `grep` para filtrar. |
| 🛑 **Finalizar un proceso** | `taskkill /PID <id>` | `kill <id>` | `kill <id>` | Termina un proceso mediante su ID (PID). En Linux/macOS se pueden usar señales como `kill -9`. |
| 📊 **Ver uso de recursos** | `Get-Process` o `systeminfo` | `top` / `htop` | `top` | Muestra estadísticas de CPU, memoria y procesos en tiempo real. |
| 🌐 **Mostrar configuración de red** | `ipconfig` | `ifconfig` o `ip a` | `ifconfig` | Muestra interfaces de red, direcciones IP y adaptadores activos. |
| 📶 **Probar conexión de red** | `ping <host>` | `ping <host>` | `ping <host>` | Envía paquetes ICMP para verificar conectividad entre hosts. |
| 🔗 **Ver rutas de red activas** | `tracert <host>` | `traceroute <host>` | `traceroute <host>` | Muestra la ruta que siguen los paquetes hasta un destino. |
| 🔒 **Cambiar contraseña de usuario** | `net user <usuario> *` | `passwd <usuario>` | `passwd <usuario>` | Permite modificar la contraseña del usuario actual o de otro. |
| ⚙️ **Información del sistema** | `systeminfo` | `uname -a` | `uname -a` | Despliega información del sistema operativo, kernel, arquitectura, etc. |
| ⏰ **Ver fecha y hora** | `time` / `date` | `date` | `date` | Muestra o modifica la fecha y hora del sistema. |
| 📦 **Instalar programas** | (Instalación manual o `winget install`) | `apt install` / `dnf install` / `yum install` | `brew install` | Instala paquetes o software desde repositorios oficiales o gestores de paquetes. |
| 🧹 **Limpiar la pantalla** | `cls` | `clear` | `clear` | Limpia la terminal y muestra una nueva línea de comandos vacía. |
| 🧭 **Salir de la terminal** | `exit` | `exit` | `exit` | Cierra la sesión actual del intérprete de comandos. |
| 🔁 **Reiniciar / Apagar** | `shutdown /r` / `shutdown /s` | `reboot` / `shutdown now` | `reboot` / `shutdown -h now` | Reinicia o apaga el sistema operativo desde la terminal. |

---

## 🧠 Notas Técnicas

- En **Linux** y **macOS**, la terminal se basa en **Unix**, por lo que comparten casi todos los comandos.
- En **Windows**, existen dos intérpretes principales:
    - **CMD** (Command Prompt): más antiguo, similar al DOS.
    - **PowerShell**: más moderno, orientado a objetos y con comandos llamados *cmdlets* (por ejemplo, `Get-Process`, `Set-Item`).
- Los comandos pueden **combinarse en scripts** (`.bat`, `.ps1`, `.sh`) para automatizar tareas del sistema.
- Linux y macOS soportan **pipes (`|`) y redirección (`>`, `>>`)**, lo que permite encadenar múltiples comandos de forma avanzada.
