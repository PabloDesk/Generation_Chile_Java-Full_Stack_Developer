# Visual Studio Code

## 1️⃣ Sobre el VSCode

[**Visual Studio Code**](https://code.visualstudio.com/) es un **editor de código** fuente desarrollado por Microsoft para Windows, Linux, macOS y Web. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código.

## 2️⃣ Guía de instalación

Primero, debemos acceder al [**enlace de descarga**](https://code.visualstudio.com/Download) del instalador y seleccionar el archivo correspondiente a nuestro sistema operativo:

![Notion/S1/media/M3.2-Download1.png](https://raw.githubusercontent.com/PabloDesk/Generation_Chile_Java-Full_Stack_Developer/refs/heads/main/Notion/S1/media/M3.2-Download1.png)

Para este caso, llevaremos a cabo la instalación en Windows. Una vez que tengamos descargado instalador, lo ejecutamos y lo primero que nos aparecerá es el acuerdo de licencia. Luego de leerlo, marcamos la casilla “Acepto el acuerdo” para poder continuar con la instalación:

![Notion/S1/media/M3.2-Install1.png](https://raw.githubusercontent.com/PabloDesk/Generation_Chile_Java-Full_Stack_Developer/refs/heads/main/Notion/S1/media/M3.2-Install1.png)

Podemos continuar con las opciones de instalación **por defecto** y en caso de ser necesario, podemos modificar la ruta en donde será instalado. Antes de instalar, se nos consultará si es que deseamos realizar tareas adicionales, podemos dejar marcadas las casillas que vienen por defecto o marcar las demás opciones según lo consideremos necesario:

![Notion/S1/media/M3.2-Install2.png](https://raw.githubusercontent.com/PabloDesk/Generation_Chile_Java-Full_Stack_Developer/refs/heads/main/Notion/S1/media/M3.2-Install2.png)

Una vez completada la instalación, se nos consulta si deseamos ejecutarla. Finalizamos el proceso de instalación.

![Notion/S1/media/M3.2-Install3.png](https://raw.githubusercontent.com/PabloDesk/Generation_Chile_Java-Full_Stack_Developer/refs/heads/main/Notion/S1/media/M3.2-Install3.png)

Finalmente, podemos abrir el Visual Studio Code y empezar a personalizar el layout, la paleta de colores y atajos. ¡Felicidades! Ya puedes comenzar a escribir código en el VSCode.

### Instalación en macOS por comandos

- Requisito recomendado: tener [Homebrew](https://brew.sh/) instalado.
1. Instalar VS Code con Homebrew:

```bash
brew install --cask visual-studio-code
```

1. Abrir VS Code desde la terminal con el comando `code`:
    - En la primera ejecución, si `code` no funciona, abre VS Code y ve a:
        
        Command Palette → “Shell Command: Install 'code' command in PATH”.
        

> Para actualizar más adelante: `brew upgrade --cask visual-studio-code`.
>

### Instalación en Linux por comandos

Selecciona el método según tu distribución.

### Debian y Ubuntu (APT)

```bash
# Importar la clave de Microsoft
wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > [packages.microsoft](http://packages.microsoft).gpg
sudo install -D -o root -g root -m 644 [packages.microsoft](http://packages.microsoft).gpg /etc/apt/keyrings/[packages.microsoft](http://packages.microsoft).gpg

# Agregar el repositorio de VS Code
sudo sh -c 'echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/[packages.microsoft](http://packages.microsoft).gpg] https://packages.microsoft.com/repos/code stable main" > /etc/apt/sources.list.d/vscode.list'

# Actualizar e instalar
sudo apt update
sudo apt install code
```

### Ubuntu con Snap

```bash
sudo snap install code --classic
```

### Fedora, RHEL, CentOS (DNF)

```bash
sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'cat > /etc/yum.repos.d/vscode.repo << EOF
[code]
name=Visual Studio Code
baseurl=https://packages.microsoft.com/yumrepos/vscode
enabled=1
gpgcheck=1
gpgkey=https://packages.microsoft.com/keys/microsoft.asc
EOF'

sudo dnf check-update
sudo dnf install code
```

### openSUSE (Zypper)

```bash
sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo zypper ar -f https://packages.microsoft.com/yumrepos/vscode vscode
sudo zypper refresh
sudo zypper install code
```

### Flatpak (alternativa universal)

```bash
flatpak install flathub com.visualstudio.code
```

> Para actualizar: usa el mismo gestor de paquetes que elegiste. Por ejemplo `sudo apt upgrade`, `sudo dnf upgrade`, `sudo zypper update`, `snap refresh` o `flatpak update`.
>


## 3️⃣ Extensiones de interés y cómo instalarlas

### ¿Para qué son las extensiones?

Las extensiones en VSCode permiten **ampliar las capacidades** de este editor, agregando más características como habilitarlo para trabajar con cierto lenguaje, temas estéticos, herramientas para depuración y para el proceso de desarrollo en general. Se puede acceder al marketplace de extensiones a través del **[siguiente enlace**](https://marketplace.visualstudio.com/)  o a través del editor.

![Notion/S1/media/M3.2-Extensions1.png](https://github.com/PabloDesk/Generation_Chile_Java-Full_Stack_Developer/blob/8e9be70022e79ec8a668591b68d3b08b97ddbd99/Notion/S1/media/M3.2-Extensions1.png)

### ¿Qué extensiones hay que instalar?

Ya sea que accedamos desde el navegador o a través del mismo VSCode, existen algunas extensiones que podemos buscar desde este momento ya que aportan funcionalidades de importancia para lo que abordaremos durante las clases, entre ellas encontramos las siguientes:

- [**Open In Browser:**](https://marketplace.visualstudio.com/items?itemName=techer.open-in-browser) nos permite abrir el archivo en el que estemos trabajando en el **navegador** por defecto, muy útill para aprender HTML, CSS y JS.

![Notion/S1/media/M3.2-OpenInBrowser.png](https://github.com/PabloDesk/Generation_Chile_Java-Full_Stack_Developer/blob/8e9be70022e79ec8a668591b68d3b08b97ddbd99/Notion/S1/media/M3.2-OpenInBrowser.png)

- [**Live Server:**](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) crea un **servidor local** para desarrollo, lo cual nos permite visualizar los cambios realizados cada vez que guardemos.

![Notion/S1/media/M3.2-LiveServer.png](https://github.com/PabloDesk/Generation_Chile_Java-Full_Stack_Developer/blob/8e9be70022e79ec8a668591b68d3b08b97ddbd99/Notion/S1/media/M3.2-LiveServer.png)

- **Para trabajar con Java:**
    - [**Extension Pack For Java:**](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack) un pack de extensiones que permiten escribir, testear y depurar código Java en el VSCode.
- **Para trabajar con Python:**
    - [Extension Pack For Python:](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack) un pack de extensiones que permiten escribir, testear y depurar código Python en el VSCode.
