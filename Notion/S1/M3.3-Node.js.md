# Node.js

## 1️⃣ Sobre Node.js

Node.js es un entorno de ejecución para JavaScript construido con el motor de JavaScript V8 de Chrome. Node.js utiliza un modelo de operaciones E/S sin bloqueo y orientado a eventos, que lo hace liviano y eficiente. El ecosistema de paquetes de Node.js, npm, es el ecosistema más grande de librerías de código abierto en el mundo.

Node.js permite desarrollar aplicaciones del lado del servidor usando JavaScript, lo que facilita crear aplicaciones web escalables y en tiempo real. A diferencia de Eclipse (que es un IDE), Node.js es un runtime environment que ejecuta código JavaScript fuera del navegador.

## 2️⃣ Guía de instalación de Node.js

### Para instalar Node.js, debemos seguir estos pasos según nuestro sistema operativo:

<aside>

**Consejo importante:** Se recomienda utilizar siempre la versión LTS (Long-Term Support) de Node.js para proyectos en producción. Las versiones LTS ofrecen mayor estabilidad, compatibilidad a largo plazo y reciben actualizaciones de seguridad durante un período más extenso, lo que las hace ideales para entornos empresariales y aplicaciones que requieren mantenimiento prolongado.

</aside>

### Windows

- Visitar el [sitio oficial de Node.js](https://nodejs.org/en/download/) y descargar el instalador para Windows
- Ejecutar el instalador y seguir las instrucciones del asistente
- En esta ocacion intalar la penultima vercion LTS (24.14.0 LTS) con chocolately.
- Verificar la instalación abriendo la terminal y ejecutando: `node --version` y `npm --version`

### macOS

- Opción 1: Descargar el instalador desde el [sitio oficial](https://nodejs.org/en/download/)
- Opción 2: Usar Homebrew: `brew install node`
- Verificar la instalación con: `node --version` y `npm --version`

### Linux (Ubuntu/Debian)

- Actualizar repositorios: `sudo apt update`
- Instalar Node.js: `sudo apt install nodejs npm`
- Verificar la instalación: `node --version` y `npm --version`

### **NVM (Node Version Manager) - (Opcional)**

<aside>

**Consejo importante:** Aunque NVM es opcional, su uso es altamente recomendable para desarrolladores. Te permite cambiar fácilmente entre diferentes versiones de Node.js según las necesidades de cada proyecto, facilita la actualización a nuevas versiones sin afectar proyectos existentes, y ayuda a evitar problemas de compatibilidad. Además, es esencial para equipos que trabajan en múltiples proyectos con diferentes requisitos de versión.
</aside>

**NVM es una herramienta esencial para desarrolladores que permite instalar y gestionar múltiples versiones de Node.js en un mismo sistema.**

### Instalación de NVM

- En Linux/macOS: `curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash`
- En Windows: Descargar [NVM for Windows](https://github.com/coreybutler/nvm-windows/releases)

### Comandos básicos de NVM

- Listar versiones disponibles: `nvm ls-remote`
- Instalar una versión específica: `nvm install 22.22.1`
- Cambiar de versión: `nvm use 22.22.1`
- Establecer versión por defecto: `nvm alias default 22.22.1`

<aside>
  
**NVM es especialmente útil en entornos de desarrollo donde diferentes proyectos requieren diferentes versiones de Node.js.**

</aside>
