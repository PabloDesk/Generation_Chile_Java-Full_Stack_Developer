# ***DBeaver***

## 1️⃣ Sobre DBeaver

DBeaver es una herramienta gratuita y de código abierto para **administrar bases de datos**. Permite conectarse a distintos motores (por ejemplo: MySQL, PostgreSQL, SQLite, SQL Server, entre otros) desde una sola interfaz.

Es muy útil en el bootcamp porque te deja:

- Ver tablas y datos de forma gráfica.
- Crear y ejecutar consultas SQL.
- Explorar esquemas y relaciones.

💡

Descarga oficial: [DBeaver Community  Download](https://dbeaver.io/download/)

Documentación: [DBeaver Documentation](https://dbeaver.com/docs/dbeaver/)

</aside>

## 2️⃣ Guía de instalación de DBeaver (Community)

Sigue los pasos según tu sistema operativo.

<aside>

**Consejo importante:** La opción recomendada es usar el **instalador oficial** (Windows/macOS) porque ya trae lo necesario y facilita las actualizaciones.[[1]](https://dbeaver.com/docs/dbeaver/Installation/)

</aside>

### **Windows**

1. Ve a la página oficial de descargas: [Download  DBeaver Community](https://dbeaver.io/download/).
2. En la sección **Windows**, descarga **Windows (installer)**.[[2]](https://dbeaver.io/download/)
3. Ejecuta el instalador (`.exe`) y avanza con las opciones por defecto.
4. Abre DBeaver desde el menú Inicio.

✅ **Verificación rápida**

- Si DBeaver abre sin errores, la instalación quedó lista.

### **macOS**

1. Ve a la página oficial de descargas: [Download  DBeaver Community](https://dbeaver.io/download/).
2. Descarga el `.dmg` según tu equipo:
    - **Apple Silicon** (M1/M2/M3).
    - **Intel**.
3. Abre el `.dmg` y arrastra DBeaver a **Applications**.
4. Abre DBeaver desde **Applications**.

⚠️ Si macOS bloquea la app por seguridad:

- Ve a **System Settings  Privacy & Security** y permite la apertura de la app.

### **Linux (Ubuntu/Debian)**

Hay varias formas de instalar. Una opción simple y común es instalarlo desde **Snap**.

1. Instala DBeaver Community:
    
    ```bash
    sudo apt-get install \
        ca-certificates \
        curl \
        gnupg
    ```
    
2. Instala los paquetes necesarios para permitir a `apt` usar un repositorio sobre HTTPS:
    
    ```bash
    sudo snap install dbeaver-ce
    ```
    
3. Abre DBeaver desde el menú de aplicaciones.

<aside>

En Linux, también existen otras distribuciones (paquetes según tu distro). Si tu instalación es diferente, revisa la documentación oficial: [Installation  DBeaver Documentation](https://dbeaver.com/docs/dbeaver/Installation/)

</aside>

## 3️⃣ Primer inicio (pasos recomendados)

Al abrir DBeaver por primera vez, es normal que:

- Te pida instalar componentes adicionales (drivers).
- Te pida confirmar descargas.

### ✅ Crear tu primera conexión (ejemplo: MySQL)

1. Abre DBeaver.
2. Selecciona **Database  New Database Connection**.
3. Elige **MySQL**.
4. Completa los datos típicos:
    - **Host:** [`localhost`](http://localhost)
    - **Port:** `3306`
    - **Database:** (opcional)
    - **User:** `root` (si aplica)
    - **Password:** tu contraseña
5. Presiona **Test Connection**.
6. Si funciona, presiona **Finish**.

<aside>
✅

Si llegas hasta el **Test Connection OK**, DBeaver quedó listo para usar.

</aside>

## 4️⃣ Problemas comunes

### 🔑 Error de conexión (MySQL): *Public Key Retrieval*

En algunos clientes (DBeaver, IntelliJ, etc.) puede aparecer este error al conectarse.

✅ **Solución típica:** agrega parámetros al URL de conexión:

- `allowPublicKeyRetrieval=true`
- `useSSL=false`

Ejemplo:

`jdbc:mysql://[localhost:3306/bootcamp?allowPublicKeyRetrieval=true&useSSL=false](http://localhost:3306/bootcamp?allowPublicKeyRetrieval=true&useSSL=false)`
