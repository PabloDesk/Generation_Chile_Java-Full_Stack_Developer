# **El Camino del Programador Feliz**

> “En el Reino del Código, solo quienes dominan la terminal encuentran el verdadero camino del programador feliz.”
> 
> 
> 🧙 *Tu misión comienza aquí…*

## 🎯 **Objetivo**

Explorar un entorno Linux dentro de Docker mientras aprendes a moverte con soltura en la terminal:

<aside>
💡

📂 crear, mover y listar archivos, 🔍 descubrir carpetas ocultas, 🔐 cambiar permisos, y 🧩 descifrar un mensaje final oculto.

</aside>

---

## ⚙️ **Requisitos previos**

- Tener **Docker Desktop** o **Docker Engine** instalado en tu computador.
    
    👉 Guía Docker: [Enlace](https://www.notion.so/3254db47a2558122be04f647da5f39ff?pvs=21)
    
- Conexión a Internet para descargar la imagen.

---

## 🚀 **1️⃣ Inicia tu aventura**

Abre una terminal (PowerShell, Git Bash o Linux) y ejecuta:

```bash
docker run -it --name linux-quest gperzal/linux-quest:1.3
```

> 🔹 Este comando descargará la imagen y abrirá tu entorno Linux dentro de Docker.
> 
> 
> 🔹 Todo el juego ocurre dentro del contenedor: ¡no afectará tu sistema real!
> 

---

## 🌍 **2️⃣ Tu entorno de juego**

Cuando ingreses verás algo como:

```
✨ Bienvenido/a a Linux Quest — “El Camino del Programador Feliz”
📍 Estás en: /home/programmer
📂 Tu misión está en: /home/programmer/quest
👉 Sugerencia: lee '0_README.txt' y 'COMANDOS_UTILES.txt'
```

Tu misión se desarrolla dentro de la carpeta:

```bash
cd ~/quest
```

---

## 🧩 **3️⃣ Primeros pasos**

1. **Lee las instrucciones iniciales:**
    
    ```bash
    cat 0_README.txt
    ```
    
2. **Explora los comandos útiles:**
    
    ```bash
    cat COMANDOS_UTILES.txt
    ```
    
3. **Descubre lo oculto 👀**
    
    ```bash
    ls -la
    ```
    
4. **Explora los caminos del bosque, el río y el castillo:**
    
    ```bash
    cd forest
    cd river
    cd castle
    ```
