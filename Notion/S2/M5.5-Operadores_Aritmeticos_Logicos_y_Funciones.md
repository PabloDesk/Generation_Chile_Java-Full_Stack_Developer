## BODMAS ****

Es un **acrónimo que indica el orden de las operaciones matemáticas para resolver correctamente una expresión**. Las letras significan: Brackets (Paréntesis), Orders (Órdenes/potencias y raíces), Division (División), Multiplication (Multiplicación), Addition (Suma) y Subtraction (Resta). Las operaciones con la misma prioridad (División/Multiplicación y Suma/Resta) se resuelven de izquierda a derecha.

### 🔢 **1. Operadores aritméticos**

| Operador | Significado | Ejemplo | Resultado |
| --- | --- | --- | --- |
| `+` | Suma | `5 + 3` | `8` |
| `-` | Resta | `10 - 2` | `8` |
| `*` | Multiplicación | `4 * 2` | `8` |
| `/` | División | `16 / 2` | `8.0` |
| `%` | Módulo (resto) | `10 % 3` | `1` |
| `**` | Potencia | `2 ** 3` | `8` |
| `//` | División entera | `17 // 2` | `8` |

> 🧩 “Los operadores no son solo símbolos: son instrucciones matemáticas que Python entiende directamente.”
> 

### 🧩 **3. Operadores lógicos básicos**

Para reforzar lógica y control:

| Operador | Significado | Ejemplo | Resultado |
| --- | --- | --- | --- |
| `and` | Verdadero si ambos lo son | `True and False` | `False` |
| `or` | Verdadero si uno lo es | `True or False` | `True` |
| `not` | Invierte el valor lógico | `not True` | `False` |

> 💬 “Los operadores lógicos se usan para tomar decisiones: por ejemplo, no dividir si el divisor es cero.”
> 

### 🧱 **4. Funciones: modularizando el pensamiento**

> [!IMPORTANT]
> “Una función es un mini programa dentro de tu programa.
> 
> Sirve para **reutilizar código** y **organizar la lógica**.”
>

Ejemplo básico:

```jsx
function saludar(nombre) {
  console.log(`Hola ${nombre}, bienvenido a JavaScript!`);
}

saludar("Soy yo");
```

Ejemplo con **retorno de valor:**

```jsx
function sumar(a, b) {
  return a + b;
}

let resultado = sumar(5, 10);
console.log("El resultado es:", resultado);

```
> [!NOTE]
> 🧠 Explica que return envía el resultado fuera de la función.
> 
> Sin `return`, la función solo ejecuta, pero no entrega un valor reutilizable.
>
