# 🧭 0) ¿Qué estamos aprendiendo realmente?

Todo programa hace SOLO 3 cosas:

1. 🧠 **Decidir** → `if`
2. 🔁 **Repetir** → `while`, `for`
3. 🧭 **Seguir un flujo** → diagramas de flujo

👉 Si entiendes esto, entiendes la base de la programación.

---

# 🗣️ 1) Traducción humana (CLAVE)

Antes de ver código, piensa así:

| Código | Significado |
| --- | --- |
| `if` | “Si pasa esto…” |
| `else` | “Si no pasa…” |
| `while` | “Repite mientras…” |
| `for` | “Repite X veces…” |

---

# 🔷 2) Diagramas de flujo (MUY IMPORTANTE)

Los diagramas ayudan a entender la lógica del programa.

❌ **NO sirven para detectar errores de sintaxis**

✅ Sirven para entender decisiones y caminos

## 🔹 Formas básicas:

| Forma | Significado |
| --- | --- |
| 🔷 Rombo | Decisión (`if`) |
| ▭ Rectángulo | Acción |
| ➡ Flecha | Flujo |
| ⭕ Óvalo | Inicio / Fin |

# ⚖️ 3) Condicionales: if / else

```jsx
let numero = 5;

if (numero % 2 === 0) {
  console.log("Par");
} else {
  console.log("Impar");
}
```

👉 Traducción:

- Si es par → imprime “Par”
- Si no → imprime “Impar”

---

## ⚠️ IMPORTANTE

El orden **SÍ importa**:

```jsx
if (x > 10) {
} else if (x > 5) {
}
```

👉 Se evalúa de arriba hacia abajo.

---

# 🔁 4) Bucle while

```jsx
let i = 1;

while (i <= 3) {
  console.log(i);
  i++; 
}
```

👉 Salida:

```
1
2
3
```

---

## ⚠️ ERROR CLÁSICO (SALE EN PRUEBA)

```jsx
let i = 1;

while (i < 7) {
  console.log(i);
}
```

❌ Esto genera un **bucle infinito**

👉 Porque `i` nunca cambia

---

# 🔁 5) Bucle for

```jsx
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

👉 Traducción:

- Empieza en 0
- Mientras sea menor a 5
- Aumenta de 1 en 1

---

# 🔁 6) for...of (recorrer listas)

```jsx
for (let i of [1, 2, 3]) {
  console.log(i);
}
```

👉 Salida:

```
1
2
3
```

---

# 🧪 7) Ejemplo paso a paso (TIPO PRUEBA)

```jsx
let i = 1;

while (i < 7) {
  if (i % 2 === 0) {
    console.log(i);
  }
  i++;
}
```

## 🔍 Paso a paso:

| i | ¿Es par? | Imprime |
| --- | --- | --- |
| 1 | ❌ | - |
| 2 | ✅ | 2 |
| 3 | ❌ | - |
| 4 | ✅ | 4 |
| 5 | ❌ | - |
| 6 | ✅ | 6 |

👉 **Salida final:**

```
2
4
6
```

---

# 🧪 8) Ejemplo con for (TIPO PRUEBA)

```jsx
for (let i of [1, 2, 3]) {
  if (i % 2 === 1) {
    console.log(i);
  }
}
```

👉 Solo imprime los impares

👉 **Salida:**

```
1
3
```

---

# ⚠️ 9) Reglas clave (APRENDER PARA LA PRUEBA)

✔ `if` sirve para tomar decisiones

✔ `while` y `for` sirven para repetir

✔ Si no cambias la variable → bucle infinito

✔ El orden de `if / else if` importa

✔ Puedes comparar textos en JavaScript

✔ `for` se puede transformar en `while`

---

# 🧠 10) Comparaciones en JavaScript

```jsx
let nombre = "Juan";

if (nombre === "Juan") {
  console.log("Son iguales");
}
```

👉 Sí se pueden comparar textos

---

# 🚨 ERRORES COMUNES

❌ Usar `==` → usar `===`

❌ Olvidar incrementar en while

❌ No entender el orden de condiciones

❌ Pensar que el diagrama detecta errores

---

# 🎯 11) Mini resumen final

👉 Un programa:

- Decide → `if`
- Repite → `while`, `for`
- Sigue un flujo → diagramas
