## ¿ Como funciona HTML ?

HTML se estructura por elementos que serán interpretados por el navegador. Cada elementos se compondrá por etiquetas y cada etiqueta puede contener ciertos atributos junto a su valor, por ejemplo, a la etiqueta <button> se le da un atributo class a la que le asignaremos su valor. Este puede ser un nombre que definamos o alguna palabra específica que definirá el comportamiento de la etiqueta, en este caso el valor es “clase-de-boton”

![Notion/S3/media/M2.10-Como_funciona_HTML.png](https://github.com/PabloDesk/Generation_Chile_Java-Full_Stack_Developer/blob/0b774d85bdca60ad3241dbc028bda6cd2d438010/Notion/S3/media/M2.10-Como_funciona_HTML.png)

Algunos de los atributos mas usados en HTML. son:

| Atributo | Descripción |
| --- | --- |
| `class` | Especifica una o más clases para un elemento. |
| `id` | Especifica un id único para un elemento. |
| `style` | Especifica estilos en línea para un elemento. |
| `src` | Especifica la fuente de una imagen, un script o un marco. |
| `href` | Especifica la URL de la página a la que enlaza un enlace. |
| `alt` | Especifica un texto alternativo para una imagen, si esta no se puede mostrar. |
| `width` y `height` | Especifican el ancho y la altura de una imagen, respectivamente. |
| `lang` | Especifica el idioma del contenido de un elemento. |

---

HTML funciona con una estructura de datos comúnmente llamada árbol o nodos. Los árboles se usan en muchas áreas de las ciencias de la computación, incluyendo sistemas operativos, gráficos, sistemas de bases de datos y redes de computadoras. Una estructura de datos tipo árbol tiene una raíz, ramas y hojas. La diferencia entre un árbol de la naturaleza y un árbol de las ciencias de la computación es que una estructura de datos tipo árbol tiene su raíz en la parte superior y sus hojas en la parte inferior.

![Notion/S3/media/M2.10-Estructura_HTML.png](https://github.com/PabloDesk/Generation_Chile_Java-Full_Stack_Developer/blob/0b774d85bdca60ad3241dbc028bda6cd2d438010/Notion/S3/media/M2.10-Estructura_HTML.png)

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <ul>
        <li>Java</li>
        <li>JavaScript</li>
    </ul>
    <h1>HackerMan</h1>
    <h2>
        <a href="https://www.youtube.com/watch?v=KEkrWRHCDQU">¡Sigue mi tutorial!</a>
    </h2>
</body>
</html>
```

La etiqueta head nos permite especificar información para el navegador, como por ejemplo,
el título o codificación del documento.

Los elementos mas utilizados en el head son los siguientes:

| Etiqueta | Descripción |
| --- | --- |
| `<title>` | Define el título del documento. |
| `<meta>` | Define metadatos del documento, como la descripción, palabras clave y codificación de caracteres. |
| `<link>` | Define relaciones entre el documento actual y otros recursos, como hojas de estilo CSS e iconos de página. |
| `<script>` | Define un script o programa que se ejecutará en el documento, como JavaScript. |
| `<style>` | Define reglas de estilo para el documento, como las propiedades de las fuentes y los colores. |

El elemento **`<body>`** en HTML es uno de los elementos fundamentales y esencial para la estructura de un documento HTML. Representa el contenido principal de un documento HTML y contiene todos los elementos que se mostrarán en la página web.

Los elementos mas utilizados en el body son los siguientes:

| Etiqueta | Descripción | Ejemplo |
| --- | --- | --- |
| `<h1>` a `<h6>` | Define los encabezados del documento, siendo `<h1>` el de mayor importancia y `<h6>` el de menor. | `<h1>Este es un título</h1>` |
| `<p>` | Define un párrafo. | `<p>Este es un párrafo.</p>` |
| `<hr>` | Es una etiqueta de auto cierre o vacía y se utiliza para crear un quiebre horizontal o línea horizontal en la página. | `h1>Sección 1</h1>` `<p>Contenido de la sección 1.</p>` `<hr> <!-- Línea horizontal -->` `<h2>Sección 2</h2>` `<p>Contenido de la sección 2.</p>p>Este es un párrafo.</p>` |
| `<a>` | Define un enlace. | `<a href="<https://www.ejemplo.com>">Este es un enlace</a>` |
| `<img>` | Define una imagen. | `<img src="imagen.jpg" alt="Descripción de la imagen">` |
| `<ul>`, `<ol>`, `<li>` | Define listas desordenadas (`<ul>`), ordenadas (`<ol>`) y sus elementos (`<li>`). | `<ul><li>Elemento 1</li><li>Elemento 2</li></ul>` |
| `<div>` | Define una sección en un documento. | `<div>Esta es una sección</div>` |
| `<span>` | Define un tramo de texto en una línea. | `<span>Este es un tramo de texto.</span>` |


**ACCESIBILIDAD EN HTML**

La accesibilidad en HTML se refiere a la práctica de diseñar y desarrollar páginas web de manera que sean accesibles para todas las personas, independientemente de sus habilidades o discapacidades. La idea es hacer que el contenido web sea comprensible y utilizable por una amplia audiencia, incluyendo personas con discapacidades visuales, auditivas, motoras o cognitivas. Aquí hay algunas buenas prácticas para mejorar la accesibilidad en HTML:

1. **Uso adecuado de encabezados (**`<h1>` a `<h6>`)**:**
- Utiliza encabezados en orden jerárquico para estructurar correctamente la información.
- Proporciona una jerarquía clara y significativa para facilitar la navegación.

```html
<h1>¡ Hola soy HackerMan !👋</h1>
<h2>Bienvenido a mi portafolio </h2>
<h3>Principales tecnologías que domino</h3>
```

2. **Atributo `alt` en imágenes:**
- Agrega un atributo **`alt`** descriptivo a todas las imágenes para que las personas con discapacidades visuales puedan comprender el contenido de la imagen.

```html
<img src="imagen.jpg" alt="Descripción de la imagen">
```

3. **Etiqueta `<label>` para formularios:**
- Utiliza la etiqueta **`<label>`** asociada a cada elemento de formulario para mejorar la navegación y comprensión del usuario.

```html
<label for="nombre">Nombre:</label>
<input type="text" id="nombre" name="nombre">
```

4. **Atributo `aria-*` para información adicional:**
- Utiliza atributos **`aria-*`** para proporcionar información adicional a tecnologías de asistencia.

```html
<div role="alert" aria-live="assertive">Mensaje de alerta importante</div>
```

5. **Enlaces descriptivos:**
- Asegúrate de que los enlaces tengan texto descriptivo, evita "clic aquí" o enlaces ambiguos.

```html
<a href="/pagina" title="Ir a la página principal">Ir a la página principal</a>
```

6. **Uso adecuado de títulos (`<title>`):**
- Proporciona títulos descriptivos y únicos para cada página.

```html
<head>
    <title>Mi Portafolio - Inicio</title>
</head>
```
