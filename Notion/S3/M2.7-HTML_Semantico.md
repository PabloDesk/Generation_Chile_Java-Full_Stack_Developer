# HTML Semántico

> “Un gran poder conlleva escribir buen HTML.” — Tío Ben, probablemente.

## 🌐 ¿Qué es HTML semántico?

HTML semántico consiste en usar **etiquetas que describen el propósito del contenido**, no solo su apariencia.

❌ “Un `<div>` para todo” → código sin sentido.

✅ “Cada etiqueta expresa su intención” → código limpio, organizado, accesible y profesional.

Ejemplos:

| No semántico | Semántico |
| --- | --- |
| `<div class="menu">` | `<nav>` |
| `<div id="header">` | `<header>` |
| `<div class="contenido">` | `<main>` |
| `<div class="seccion">` | `<section>` |
| `<div class="noticia">` | `<article>` |
| `<div class="pie">` | `<footer>` |

## 🧩 ¿Por qué es importante?

✔️ Mejora la **accesibilidad**

✔️ Mejora la **lectura del código**

## 🧠 HTML Semántico en HTML5 (lo esencial)

Estas etiquetas deben ser parte natural de tu HTML:

- `<header>`
- `<nav>`
- `<main>`
- `<section>`
- `<article>`
- `<aside>`
- `<footer>`
- `<figure>` + `<figcaption>`
- `<table>` con `<thead>`, `<tbody>`, `<tfoot>`

---

# 🔽 **Las Etiquetas Semánticas Explicadas**

### 🔽 `<header>`

Encabezado general del sitio o de una sección.

```html
<header>
  <h1>The Javengers</h1>
  <nav>…</nav>
</header>
```

---

### 🔽 `<nav>`

Zona de navegación del sitio.

```html
<nav>
  <ul>
    <li><a href="#heroes">Héroes</a></li>
    <li><a href="#misiones">Misiones</a></li>
    <li><a href="#contacto">Contacto</a></li>
  </ul>
</nav>
```

---

### 🔽 `<main>`

Contenido central y más importante de la página.

```html
<main>
  <section>…</section>
</main>
```

---

### 🔽 `<section>`

Bloques temáticos de contenido.

```html
<section id="misiones">
  <h2>Misiones activas</h2>
</section>
```

---

### 🔽 `<article>`

Contenido independiente y reutilizable.

```html
<article>
  <h3>Reporte de misión</h3>
  <p>Victoria absoluta del equipo C22.</p>
</article>
```

---

### 🔽 `<aside>`

Información complementaria.

```html
<aside>
  <p>Dato curioso: 87% de los Javengers prefieren VSCode.</p>
</aside>
```

---

### 🔽 `<footer>`

El final del documento o sección.

```html
<footer>
  <p>© 2025 C22 The Javengers</p>
</footer>
```

---

## 🎯 Buenas prácticas en HTML semántico

      ✔Solo un `<h1>` por página

✔ Ordenar títulos correctamente (H1 → H2 → H3…)

✔ No usar `<br>` para separar secciones

✔ No usar `<div>` para todo (divitis)

✔ Usar etiquetas semánticas siempre que exista una apropiada

✔ Usar textos alternativos (`alt=""`) en imágenes

✔ Organizar el proyecto en carpetas (`css/`, `js/`, `img/`)

✔ Formularios accesibles (usar `<label for="">`)

✔ Escribir **meta tags** en el `<head>`

# `js/script.js` (Interacción con el DOM)

```jsx
// Cuando el DOM esté listo
document.addEventListener("DOMContentLoaded", () => {
  console.log("🌐 C22 – The Javengers: página semántica cargada.");

  // Scroll suave a la sección de misiones
  const btnScrollMisiones = document.getElementById("btnScrollMisiones");
  const seccionMisiones = document.getElementById("misiones");

  if (btnScrollMisiones && seccionMisiones) {
    btnScrollMisiones.addEventListener("click", () => {
      seccionMisiones.scrollIntoView({ behavior: "smooth" });
    });
  }

  // Lógica de misiones completadas
  let misionesCompletadas = 0;
  const spanMisiones = document.getElementById("misionesCompletadas");
  const botonesMision = document.querySelectorAll(".btn-mission");

  botonesMision.forEach((btn) => {
    btn.addEventListener("click", () => {
      // Si ya está completada, no suma de nuevo
      if (!btn.classList.contains("completed")) {
        btn.classList.add("completed");
        btn.textContent = "Completada ✅";
        misionesCompletadas++;
        if (spanMisiones) {
          spanMisiones.textContent = misionesCompletadas;
        }
      }
    });
  });

  // Manejo simple del formulario de contacto
  const contactForm = document.getElementById("contactForm");
  const formFeedback = document.getElementById("formFeedback");

  if (contactForm && formFeedback) {
    contactForm.addEventListener("submit", (event) => {
      event.preventDefault(); // Evita recargar la página

      const nombre = contactForm.nombre.value.trim();
      const email = contactForm.email.value.trim();

      if (!nombre || !email) {
        formFeedback.textContent =
          "Por favor, completa al menos tu nombre y correo. 💬";
        formFeedback.style.color = "#ffdd57";
        return;
      }

      formFeedback.textContent =
        "¡Gracias por tu mensaje, Javenger! Nos pondremos en contacto pronto. 🦾";
      formFeedback.style.color = "#00d4ff";

      // Reset opcional
      contactForm.reset();
    });
  }
});
```
