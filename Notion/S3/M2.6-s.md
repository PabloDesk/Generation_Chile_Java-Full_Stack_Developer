# Etiquetas HTML

## Etiquetas de Estructura del Documento

| Etiqueta | Definición | Uso |
| --- | --- | --- |
| `<!DOCTYPE html>` | Declaración del tipo de documento HTML5 | Siempre al inicio del documento para indicar que es HTML5 |
| `<html>` | Elemento raíz del documento | Contiene todo el contenido de la página web |
| `<head>` | Encabezado del documento | Contiene metadatos, enlaces a CSS, scripts y título |
| `<title>` | Título de la página | Define el título que aparece en la pestaña del navegador |
| `<body>` | Cuerpo del documento | Contiene todo el contenido visible de la página |
| `<meta>` | Metadatos | Define metadatos como charset, viewport, descripción, keywords |
| `<link>` | Enlace a recursos externos | Vincula hojas de estilo CSS y otros recursos |
| `<style>` | Estilos CSS internos | Contiene código CSS dentro del HTML |
| `<script>` | Scripts JavaScript | Incluye o referencia código JavaScript |

## Etiquetas Semánticas de Estructura (HTML5)

| Etiqueta | Definición | Uso |
| --- | --- | --- |
| `<header>` | Encabezado de sección | Contiene contenido introductorio, logo, navegación |
| `<nav>` | Navegación | Define un conjunto de enlaces de navegación |
| `<main>` | Contenido principal | Engloba el contenido principal único de la página |
| `<article>` | Artículo independiente | Contenido autocontenido como posts de blog, noticias |
| `<section>` | Sección temática | Agrupa contenido relacionado temáticamente |
| `<aside>` | Contenido complementario | Contenido tangencial como barras laterales |
| `<footer>` | Pie de página | Información de cierre, copyright, enlaces adicionales |
| `<figure>` | Contenido ilustrativo | Envuelve imágenes, diagramas, código con leyenda |
| `<figcaption>` | Leyenda de figura | Proporciona descripción para `<figure>` |

## ⚠️ ETIQUETAS DEPRECADAS (NO USAR)

Estas etiquetas ya no son válidas en HTML5 y deben evitarse:

| Etiqueta | Definición | Alternativa Moderna |
| --- | --- | --- |
| `<acronym>` | Acrónimo | Usar `<abbr>` |
| `<applet>` | Java applet | Usar `<object>` o `<embed>` |
| `<basefont>` | Fuente base | Usar CSS |
| `<big>` | Texto grande | Usar CSS (font-size) |
| `<center>` | Centrar texto | Usar CSS (text-align: center) |
| `<dir>` | Lista de directorio | Usar `<ul>` |
| `<font>` | Fuente del texto | Usar CSS (font-family, color, size) |
| `<frame>` | Marco | Evitar o usar `<iframe>` si es necesario |
| `<frameset>` | Conjunto de marcos | No usar, diseño con CSS |
| `<noframes>` | Sin marcos | No necesario en HTML5 |
| `<strike>` | Tachado | Usar `<del>` o `<s>`, o CSS |
| `<tt>` | Texto de teletipo | Usar `<code>` o CSS (font-family: monospace) |
| `<marquee>` | Texto en movimiento | Usar CSS animations o JavaScript |
| `<blink>` | Texto parpadeante | Usar CSS animations |

## 📝 Notas Importantes

### Atributos Globales Más Usados

- `class`: Asigna una o más clases CSS
- `id`: Identificador único del elemento
- `style`: Estilos CSS en línea
- `title`: Información adicional (tooltip)
- `data-*`: Atributos de datos personalizados
- `aria-*`: Atributos de accesibilidad
- `hidden`: Oculta el elemento
- `lang`: Idioma del contenido

### Buenas Prácticas 2025-2026

1. **Usar etiquetas semánticas** para mejorar SEO y accesibilidad
2. **Evitar etiquetas deprecadas** y usar CSS para estilos
3. **Incluir atributos `alt` en imágenes** para accesibilidad
4. **Usar `<button>` en lugar de `<input type="button">`** cuando sea posible
5. **Estructurar correctamente** con header, nav, main, section, aside, footer
6. **Validar HTML** con el validador de W3C
7. **Optimizar para dispositivos móviles** con meta viewport
8. **Implementar Progressive Enhancement** pensando primero en HTML funcional

### Etiquetas Modernas a Destacar

- `<dialog>`: Nativo para modales (cada vez mejor soportado)
- `<details>` y `<summary>`: Acordeones nativos sin JavaScript
- `<picture>`: Imágenes responsive con art direction
- `<template>`: Para Web Components y frameworks modernos
- `<slot>`: Fundamental en Web Components
