# 🧱 **1. HTML – La estructura del sitio web**

HTML es el **esqueleto** de toda página web.

Define la estructura, el contenido y la jerarquía de los elementos.

### Conceptos clave:

- HTML organiza el contenido mediante **elementos**.
- Cada elemento tiene:
    - **Etiqueta de apertura**
    - **Contenido**
    - **Etiqueta de cierre**
- Existen **etiquetas vacías** (como `<img>`).
- Los elementos pueden tener **atributos**, como:
    - `id` → identifica de forma única un elemento.
    - `src` → indica origen de una imagen.
- HTML forma un **árbol jerárquico** donde cada elemento es un **nodo**.

**Para qué sirve HTML:**

Es la base del contenido y estructura. Sin HTML, nada se muestra.

# 🎨 **2. CSS – El estilo y la apariencia**

CSS es la **piel** del sitio web.

Permite dar forma, colores, tamaños, tipografías y diseño visual.

### Conceptos clave:

- Se usa mediante:
    - `<style>` en el mismo archivo
    - `<link>` para cargar un archivo CSS externo
- CSS funciona con:
    - **Selectores:** `elemento`, `.clase`, `#id`
    - **Declaraciones:** `propiedad: valor;`
- Las clases agrupan elementos.
- Los IDs identifican elementos únicos.

**Para qué sirve CSS:**

Hace que el sitio sea visualmente claro, agradable e intuitivo.

# ⚡ **3. JavaScript – La interacción y dinamismo**

JavaScript es lo que vuelve la web **dinámica** y **activa**.

Permite cambiar contenido, mover elementos, responder a acciones del usuario y manipular el DOM.

### Conceptos clave:

- JS se ejecuta en el navegador (Client-Side).
- Interactúa con HTML mediante el **DOM**.
- Cada elemento HTML es un **nodo** del DOM.
- Métodos fundamentales:
    - `document.getElementById()`
    - `.innerHTML` para cambiar contenido
    - `.style` para modificar estilos
- Para usar JS en HTML existen dos formas:
    - JavaScript **interno** (dentro de `<script>`).
    - JavaScript **externo** (archivo `.js` con `src="..."`).

**Para qué sirve JavaScript:**

Hace que una página funcione en tiempo real: botones, menús, animaciones, actualizaciones sin refrescar la página.

# 🔗 **4. Cómo trabajan juntos (HTML + CSS + JS)**

Toda página web moderna combina estos tres pilares:

| Tecnología | Rol |
| --- | --- |
| **HTML** | Qué se muestra |
| **CSS** | Cómo se ve |
| **JavaScript** | Cómo se comporta |

**Ejemplo simple mental:**

Un sitio es como una casa:

- HTML = la estructura y paredes
- CSS = la decoración y estilo
- JS = la electricidad, puertas automáticas, interacciones

# 🧭 **5. Por qué estos fundamentos son esenciales para un desarrollador**

Un buen desarrollador debe dominar estos fundamentos porque:

- Todos los sitios usan HTML y CSS.
- Las funciones modernas dependen de JavaScript.
- Son la base para frameworks más avanzados como React, Angular, Vue o Svelte.
- Permiten comprender cómo trabaja realmente el navegador.
- Son necesarios para crear interfaces reales, atractivas y funcionales.
