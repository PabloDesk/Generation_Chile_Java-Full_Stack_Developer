# **1️⃣ ¿Qué es el Desarrollo Web Frontend?**

El frontend es la parte visual y dinámica de una aplicación web. Aquí se combinan los tres pilares:

| Tecnología | ¿Para qué sirve? | Ejemplo de uso |
| --- | --- | --- |
| **HTML** | Estructura | Títulos, párrafos, botones, formularios |
| **CSS** | Estilos | Colores, tamaños, layouts responsivos |
| **JavaScript** | Interactividad | Clicks, animaciones, validaciones, eventos |

💡 Hoy, en 2025–2026, todo desarrollador full-stack debe dominar estos fundamentos antes de avanzar a frameworks modernos.

---

# **2️⃣ HTML — La estructura del sitio**

### 🔹 Elementos esenciales que sí o sí deben conocer:

- `<h1>–<h6>`: encabezados
- `<p>`: párrafos
- `<img>`: imágenes
- `<button>`: botones
- `<span>`: estilizado de palabras específicas
- `<div>`: contenedores generales

Ejemplo:

```html
<h1>Hola Mundo 🌍</h1>
<p>Bienvenido a tu primera página web</p>
<img src="./images/cat.png" alt="Cute Cat" />
```

---

# **3️⃣ CSS — Cómo darle estilo moderno a tu UI**

### Formas comunes de conectar CSS con HTML:

| Tipo | Ejemplo | Cuándo usarlo |
| --- | --- | --- |
| **Archivo externo** | `<link rel="stylesheet" href="style.css">` | Mejor práctica |
| **Inline** | `<p style="color:red">` | Nunca recomendado |
| **Interno** | `<style> p {color:blue;} </style>` | Prototipos rápidos |

### Ejemplo básico:

```css
body {
  background-color: #f4f4f4;
  font-family: sans-serif;
}

h1 {
  color: #0056ff;
}
```

# **4️⃣ JavaScript — Añadiendo vida real a tus páginas**

Aquí es donde el sitio pasa de “estático” a “interactivo”.

### Funciones y métodos esenciales de esta unidad:

---

## **🔹 4.1 alert() – Mostrar mensajes emergentes**

```jsx
alert("🚨 Atención: sesión por expirar");

```

Usos típicos en 2025:

- Avisos de seguridad
- Alertas rápidas de formulario
- Confirmaciones

---

## **🔹 4.2 console.log() – Tu mejor amigo para depurar**

```jsx
console.log("Cargando datos del usuario...");
```

Sirve para:

- Ver valores de variables
- Revisar errores
- Aprender a programar leyendo la salida en consola

---

## **🔹 4.3 onclick – Activar funciones al hacer clic**

```html
<button onclick="sayHello()">Haz clic aquí</button>
```

```jsx
function sayHello() {
  console.log("Hola desde la función 👋");
}
```

---

## **🔹 4.4 El elemento <span> – Selección específica dentro de un texto**

Permite manipular *solo una parte* del contenido:

```html
<p>Las manzanas son <span id="red">rojas</span></p>
```

```jsx
document.getElementById("red").style.color = "red";
```

---

# **5️⃣ Interacción con el DOM (versión moderna y simple)**

DOM = Document Object Model.

Es la representación en memoria del HTML.

Comando más usado:

```jsx
document.getElementById("mensaje").innerText = "Nuevo texto!";
```

Acciones típicas:

| Acción | Ejemplo |
| --- | --- |
| Cambiar color | `.style.color = "blue"` |
| Cambiar texto | `.innerText = "Hola"` |
| Leer valores | `.value` |
| Ocultar/mostrar | `.style.display = "none"` |

---

# **6️⃣ Cómo conectar HTML + CSS + JS correctamente**

### Estructura mínima recomendada:

```
project/
 ├── index.html
 ├── style.css
 └── script.js

```

### index.html

```html
<script src="script.js" defer></script>

```

**Por qué `defer` es moderno y recomendado (2025–2026):**

- Evita errores de “JS cargó antes que el HTML”
- Acelera performance
- Es la forma estándar hoy
