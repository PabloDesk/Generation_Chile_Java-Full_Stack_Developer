# **Metadatos**

## 🧠 1. Estructura base donde van los metadatos

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <!-- Aquí van los metadatos -->
</head>
<body>
</body>
</html>
```

---

## 🧾 2. Metadatos básicos esenciales

### ✅ 1. Charset (codificación de caracteres)

Define cómo se interpretan los caracteres.

```html
<meta charset="UTF-8">
```

✔ Recomendado: UTF-8

✔ Evita errores con tildes, ñ, símbolos

📌 Fuente: MDN Web Docs – `<meta charset>`

---

### ✅ 2. Viewport (responsive design)

Permite que la página se adapte a móviles.

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

✔ Fundamental para diseño responsive

📌 Fuente: Google Developers – Mobile Friendly Guidelines

---

### ✅ 3. Title (título de la página)

Es lo que aparece en la pestaña del navegador y en Google.

```html
<title>Mi Página Web</title>
```

✔ Muy importante para SEO

📌 Fuente: MDN – `<title>`

---

### ✅ 4. Description (descripción SEO)

Resumen de la página para buscadores.

```html
<meta name="description" content="Esta es una página de ejemplo para aprender HTML.">
```

✔ Influye en cómo aparece en resultados de búsqueda

📌 Fuente: Google Search Central

---

### ✅ 5. Keywords (palabras clave) *(uso limitado hoy)*

Antes era relevante para SEO, hoy casi no se usa.

```html
<meta name="keywords" content="html, programación, web">
```

⚠ Google ya no lo considera relevante

📌 Fuente: Google (2009, confirmado en documentación SEO)

---

### ✅ 6. Author (autor de la página)

```html
<meta name="author" content="Hackerman">
```

✔ Útil para documentación y proyectos

📌 Fuente: MDN – `<meta name="author">`

---

## 🌐 3. Metadatos para redes sociales (Open Graph)

Usados por Facebook, LinkedIn, etc.

```html
<meta property="og:title" content="Mi Página Web">
<meta property="og:description" content="Aprendiendo HTML básico">
<meta property="og:image" content="https://ejemplo.com/imagen.jpg">
<meta property="og:url" content="https://ejemplo.com">
```

✔ Controlan cómo se ve el link al compartirlo

📌 Fuente: Open Graph Protocol (ogp.me)

---

## 🐦 4. Metadatos para Twitter

```html
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Mi Página Web">
<meta name="twitter:description" content="Aprendiendo HTML">
```

📌 Fuente: Twitter Developer Docs

---

## 📦 5. Ejemplo completo (mínimo recomendado)

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <title>Mi Página Web</title>
    
    <meta name="description" content="Página de ejemplo para aprender HTML">
    <meta name="author" content="Hackerman">

    <!-- Open Graph -->
    <meta property="og:title" content="Mi Página Web">
    <meta property="og:description" content="Aprendiendo HTML básico">
    <meta property="og:type" content="website">
</head>
<body>
</body>
</html>
```

---

## Resumen rápido (para tus alumnos)

| Meta | Para qué sirve |
| --- | --- |
| charset | Evitar errores de texto |
| viewport | Responsive (móvil) |
| title | Nombre en navegador / SEO |
| description | Descripción en Google |
| author | Autor |
| og:* | Compartir en redes |
